## 1. Общий стиль

### Тон общения:
- **Дружелюбный** — тёплое приветствие, позитивный настрой
- **Профессиональный** — строительная терминология, но с пояснениями
- **Информативный** — конкретные ответы, без воды
- **Не повторять** информацию клиента другими словами
- При перечислении — всегда **список**, не сплошной текст

### Фирменный стиль:
- Соответствие бренду SIA «MULTISPA» / betonaveidni.lv
- Цветовая гамма: тёмно-синий (#1B3A5C), оранжевый (#E87722), белый (#FFFFFF)
- Логотип компании в виджете на сайте

| Тип | Когда используется |

### 3.2 🌐 Веб-виджет — UI дизайн

**Расположение:**
- Виджет-кнопка в **правом нижнем углу** сайта
- При нажатии открывается панель чата
- Адаптивный: десктоп (350×500 px) и мобильная версия (100% ширины)

**Элементы виджета:**

```
┌──────────────────────────────┐
│  🏗️ SIA «MULTISPA»     ✕    │  ← Заголовок + закрыть
│  Виртуальный ассистент       │
├──────────────────────────────┤
│                              │
│  🤖 Сообщение бота           │  ← Пузырь (серый фон)
│                              │
│         Сообщение клиента 👤 │  ← Пузырь (синий фон)
│                              │
│  ┌────────┐ ┌────────┐      │  ← Быстрые кнопки
│  │ Аренда │ │ Покупка│      │
│  └────────┘ └────────┘      │
│                              │
├──────────────────────────────┤
│  [Введите сообщение...] 📎 ▶│  ← Поле ввода + файл + отправить
└──────────────────────────────┘
```

**Стили:**
- Шрифт: Inter / Roboto
- Сообщения бота: фон `#F0F0F0`, текст `#333`
- Сообщения клиента: фон `#1B3A5C`, текст `#FFF`
- Кнопки: border `#E87722`, текст `#E87722`, hover fill `#E87722` + текст `#FFF`
- Скругление: 12px для пузырей, 8px для кнопок
- Анимация появления: fade-in 0.3s

**Кнопка вызова виджета:**
```
┌──────┐
│  💬  │  ← Круглая кнопка, 60×60 px
└──────┘    Цвет: #E87722, тень, пульсация при первом посещении
```

### 3.3 📞 Телефон — голосовой дизайн

**Приветствие (IVR):**
```
«Sveicināti! Jūs zvanāt SIA MULTISPA.
 Здравствуйте! Вы позвонили в SIA MULTISPA.
 Hello! You have reached SIA MULTISPA.

 Lūdzu, sakiet valodu, kurā vēlaties sazināties.
 Пожалуйста, скажите, на каком языке вы хотите общаться.
 Please say which language you'd like to communicate in.»
```

**Правила голосового стиля:**
- Голос: нейтральный, профессиональный
- Темп: средний, чёткая артикуляция
- Паузы: 1,5 сек между предложениями
- Повтор: если не распознано — переспросить максимум 2 раза
- Fallback: «Для связи со специалистом, пожалуйста, оставайтесь на линии»
