# 🚀 Этап 5. Запуск продукта локально

> **Цель:** Сгенерировать код и запустить продукт на своём компьютере.

## Инструкция по запуску:
1. Скопируй `.env.example` → `.env`
2. Заполни переменные окружения (BOT_TOKEN, SUPABASE_URL, SUPABASE_KEY)
3. Установи зависимости: `npm install`
4. Запусти бота: `npm run dev`
